MySQL - 基础语法

SQL

Structured Query Language：结构化查询语言
定义了了操作所有关系型数据库的规则

SQL 通用语法

  1. SQL 语句可以单行或多行书写，以分号结尾
  2. 可使用空格和缩进来增强语句的可读性
  3. MySQL 数据库的SQL语句不区分大小写，关键字建议大写
  4. 注释：
    1. 单行注释：-- 注释内容 或 # 注释内容 （MySQL独有）
    2. 多行注释：/* 注释内容 */
